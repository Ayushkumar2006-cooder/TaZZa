# 🍽️ TaZZa – Restaurant Website

## 📖 Overview

**TaZZa** is a responsive restaurant website developed as part of my **2nd Semester BCA Web Technology Project**. The project was created to strengthen my understanding of frontend web development by building a multi-page website with an attractive user interface and responsive design.

The website showcases a modern restaurant with dedicated sections for products, gallery, about, and contact information while focusing on clean design, easy navigation, and an engaging user experience.

---

## 🎯 Project Objectives

- Develop a complete multi-page website using HTML, CSS, and JavaScript.
- Apply responsive web design principles.
- Improve frontend development skills through practical implementation.
- Create an organized and visually appealing website.
- Understand website navigation and user experience design.

---

## ✨ Features

- Responsive layout for desktop, tablet, and mobile devices
- Interactive homepage
- About Us page
- Product/Menu section
- Image gallery with Lightbox functionality
- Contact page
- Custom 404 Error Page
- Smooth navigation between pages
- Clean and organized UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Lightbox2 Library


## 📋 Website Pages

### 🏠 Home
Introduces the restaurant and highlights featured content.

### 👨‍🍳 About
Provides information about the restaurant, its story, and its services.

### 🍕 Products
Displays the available food items in an organized layout.

### 🖼️ Gallery
A collection of restaurant and food images with Lightbox support for an enhanced viewing experience.

### 📞 Contact
Allows visitors to find contact details and connect with the restaurant.

### ⚠️ Custom 404 Page
Displays a user-friendly error page when a requested page cannot be found.
